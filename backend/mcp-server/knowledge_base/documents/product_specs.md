## 4. 产品选型指南（按场景推荐）

1. **建筑外墙常规节能项目**：优先 RB95，常用厚度 50/60/80mm；若风荷载较高或锚固密度大，升级至 RB120。  
2. **高层与重点防火区域**：优先 RS95 或 RS120 做防火隔离带，厚度建议与外墙主保温层一致，减少节点热桥。  
3. **工业设备与风管保温**：优先 FA95 或 FA120；高湿或有冷凝风险场景选贴铝箔系列并强化搭接密封。  
4. **屋面与受压较大部位**：优先 RB120/RB155，厚度 80-120mm，兼顾抗压与长期尺寸稳定。  
5. **成本敏感且工况温和项目**：可选 RB80 或 RS80，但需校核强度与耐久性是否满足设计要求。  
6. **厚度选择经验**：寒冷地区优先 80-120mm，温和地区常用 50-80mm，最终以热工计算和地方节能标准为准。

## 6. 产品性能对比说明（不同容重差异）

- **保温性能**：容重从 80 提升到 120 时，导热系数通常小幅优化，但并非线性下降；过高容重不一定继续改善保温。  
- **力学性能**：容重越高，抗压和尺寸稳定性一般越好，适合高风压、机械固定和长期振动工况。  
- **施工与经济性**：低容重材料更轻、搬运便利、成本友好；高容重材料施工容错更高但材料成本增加。  
- **耐久与系统匹配**：高容重在节点、洞口和高应力区域优势明显，建议与锚固方案、抹面系统和防火构造协同设计。  
- **选型原则**：先满足防火与强度，再优化导热与成本，最终以系统性能达标和全寿命周期成本最低为目标。
