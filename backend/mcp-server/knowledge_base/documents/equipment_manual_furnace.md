# TY-MF1500 型冲天炉操作手册

## 1. 设备概述

### 1.1 设备信息
- 设备名称：熔炉（冲天炉）
- 设备型号：TY-MF1500
- 制造厂家：TY Thermal Equipment Co., Ltd.
- 产线位置：岩棉生产线“原料准备”后、“离心成纤”前的核心热工段
- 主要作用：将玄武岩、辉绿岩、白云石等配合料稳定熔融为可成纤熔体，为下游离心成纤提供连续、均匀、黏度可控的熔液

### 1.2 工艺定位与作用说明
TY-MF1500 冲天炉是整条岩棉线的“热源与质量源头”。熔融状态直接决定后续纤维直径分布、渣球含量和产品导热性能。该炉型采用燃气燃烧与强制送风的联动控制方式，依靠分区测温、燃气流量调节和炉压管理实现熔融窗口稳定。操作员应理解“温度达标不等于熔液合格”，必须同时关注熔液流动性、均匀性、出液连续性和单位能耗。

### 1.3 主要技术参数表

| 参数项 | 标准范围 | 设计/控制目的 |
|---|---|---|
| 熔融温度 | 1400-1500℃ | 形成可成纤黏度窗口 |
| 燃气流量 | 100-140 m³/h | 保证稳定热输入 |
| 风压 | 0.3-0.4 MPa | 维持燃烧强度与换热效率 |
| 熔融时间 | 4-6 h | 保障充分熔融与均化 |
| 出液稳定性 | 连续、无明显脉动 | 满足离心机连续供液 |
| 炉压控制 | 微负压运行 | 降低漏风、提升热效率 |

## 2. 操作规程

### 2.1 开机前检查清单
1. 原料系统：配料单、批次标签、粒度与含水率记录齐全。
2. 炉体与耐火层：观察孔、炉门、流口处无明显裂纹和掉砖。
3. 能源系统：燃气总阀、支路阀、调压阀、流量计状态正常，无泄漏报警。
4. 送风系统：鼓风机、风阀、风压表正常，风道无堵塞。
5. 测控系统：热电偶、压力变送器、流量计在线，DCS 趋势画面可用。
6. 安全系统：可燃气体报警器、联锁、紧急切断阀、灭火器材有效。
7. 交接确认：上班残留工况、检修票、工艺偏离记录已闭环。

### 2.2 启动步骤
1. 启动引风与送风系统，确认炉膛负压建立。
2. 点火前执行燃气管线吹扫，检测可燃气浓度满足点火条件。
3. 按点火程序启动主燃烧器，先小火稳定，再逐步升至工艺负荷。
4. 按升温曲线分阶段升温（预热—过渡—熔融—均化），禁止跨段猛升。
5. 温度进入 1400℃ 以上后，观察熔池状态，调整燃气与风压至稳定窗口。
6. 进入出液准备，确认流口状态与下游离心机联锁条件满足后放液。

### 2.3 正常运行监控要点
- 每 10-15 分钟巡检一次温度、燃气、风压、炉压趋势。
- 观察出液是否连续，是否有断流、溅流、结瘤迹象。
- 对比单位时间燃气消耗与出液量，识别异常高耗能状态。
- 记录熔液外观（亮度、均匀性、杂质）与下游成纤反馈（渣球、纤维粗细）。
- 发生参数偏离时优先小幅联动调整，严禁单参数大跳变。

### 2.4 停机步骤
1. 逐步降低加料与燃气负荷，保持风量匹配，避免局部过热。
2. 按降温曲线执行阶梯降温，避免耐火材料热冲击。
3. 停止主燃烧后保留必要引风，排净可燃气残留。
4. 隔离燃气总阀并挂牌，确认炉压与温度降至安全区间。
5. 完成停机点检：炉衬、流口、风道、仪表状态记录归档。

## 3. 参数标准

### 3.1 参数标准范围

| 监控参数 | 标准范围 | 偏离影响 |
|---|---|---|
| 熔融温度 | 1400-1500℃ | 低于下限易未熔颗粒，高于上限易过烧和能耗上升 |
| 燃气流量 | 100-140 m³/h | 过低热量不足，过高导致浪费与局部高温 |
| 风压 | 0.3-0.4 MPa | 过低燃烧不稳，过高易扰动炉内热场 |
| 熔融时间 | 4-6 h | 过短均化不足，过长效率下降 |
| 炉压 | 微负压 | 正压会漏烟，负压过大引入冷风 |

### 3.2 温度控制策略
- 以“分段斜率+分区温差”双控制为原则：预热期重在安全，熔融期重在稳定，均化期重在一致。
- 温度微调优先在 ±10-20℃ 小幅迭代，并观察 10-20 分钟再决策下一步。
- 下游出现渣球上升时，先验证熔液均化，再考虑抬温。

### 3.3 燃气调节方法
1. 确认当前燃气、风压和氧量（若有）联动状态；
2. 每次调整建议步长 2-5 m³/h；
3. 调整后核查火焰形态、温度梯度和出液连续性；
4. 若出现温升慢但耗气高，优先检查漏风与炉衬状况；
5. 参数恢复后更新班组经验表。

### 3.4 升温/降温曲线执行要点
- 升温必须遵循“慢—快—稳”原则，防止结拱、塌料、热应力开裂。
- 降温遵循“分段缓降”，严禁骤停鼓风后立即全关，防止回火和炉衬龟裂。
- 每次异常停炉后必须复核曲线执行记录，必要时调整次日启动方案。

## 4. 常见故障与处理

1. **故障现象→** 熔液中未熔颗粒明显增多  
   **可能原因→** 原料粒度超标、含水率高、熔融时间不足  
   **处理方法→** 复筛原料，延长均化时间，按窗口补偿温度  
   **预防措施→** 严格来料验收与粒度控制，执行配料前含水检测

2. **故障现象→** 出液忽快忽慢，离心机供液波动  
   **可能原因→** 流口结瘤、炉温波动、燃气压力脉动  
   **处理方法→** 清理流口结瘤，稳定燃气压力，联动修正风压  
   **预防措施→** 定时巡检流口，保持燃气调压系统维护

3. **故障现象→** 温度达标但渣球率升高  
   **可能原因→** 熔液均化不足、成分波动、局部过热  
   **处理方法→** 增加均化停留，核查配比批次，优化分区控温  
   **预防措施→** 班次配方复核，建立温度-质量关联模型

4. **故障现象→** 燃气单耗异常升高  
   **可能原因→** 炉体漏风、耐火层损伤、空燃比失调  
   **处理方法→** 排查漏风点并封堵，修补炉衬，重整空燃比  
   **预防措施→** 周期热像巡检，月度能耗对标分析

5. **故障现象→** 炉门周边冒烟或回火风险增加  
   **可能原因→** 炉压失控、引风不足、门封老化  
   **处理方法→** 调整引风与风阀，恢复微负压，更换门封  
   **预防措施→** 关键密封件按寿命计划更换，联锁定检

6. **故障现象→** 耐火层局部剥落  
   **可能原因→** 热冲击频繁、侵蚀加速、材料寿命到期  
   **处理方法→** 停炉后分区修补或更换，复核升降温策略  
   **预防措施→** 严格执行升降温曲线，年度大修更换关键区域衬体

## 5. 维护保养

### 5.1 日常保养项目清单
- 检查燃气阀组、法兰、软连接泄漏情况。
- 检查测温点偏差与波动，核对现场与DCS读数一致性。
- 清理炉前作业区域，保持观察口、检修口可视与可操作。
- 记录当班能耗、温度曲线、异常事件和处置结果。

### 5.2 周保养
- 检查流口、炉门、密封垫磨损；必要时局部修补。
- 校验燃气流量计、风压表零点与响应。
- 检查鼓引风机轴承与联轴器状态，补充润滑。

### 5.3 月保养
- 对耐火材料做可视检查与热像诊断，评估热损失区域。
- 复核空燃比控制逻辑，优化节能工况。
- 检测联锁与报警功能（超温、低压、燃气泄漏等）。

### 5.4 年度检修内容
- 重点更换炉喉、流口、燃烧器周边高损耗耐火层。
- 对燃气调压与安全切断系统进行整套校验。
- 对风机、电机、阀门执行大修与精度恢复。
- 完成年度热工评估：能耗基线、温度均匀性、故障统计与改造建议。

### 5.5 耐火材料检查与更换周期建议
- 炉门、流口高冲刷区：每月重点检查，6-12 个月评估更换。
- 炉墙普通区：季度检查，12-24 个月按劣化程度分批更换。
- 遇异常停炉、温度失控或侵蚀加剧时，立即触发临时检修。
