# TY-LX6000 型四辊离心机操作手册

## 1. 设备概述

### 1.1 设备信息
- 设备名称：四辊离心成纤机
- 设备型号：TY-LX6000
- 制造厂家：TY Fiber Machinery Co., Ltd.
- 产线位置：熔炉出液之后、集棉铺毡之前
- 主要作用：将高温熔体通过高速离心与气流牵伸形成细纤维，决定岩棉纤维细度、渣球率和后续成毡质量

### 1.2 工艺作用说明
TY-LX6000 是岩棉产线“纤维形成核心设备”。设备由四辊甩丝系统、驱动系统、轴承润滑冷却系统、振动监测系统和安全联锁系统组成。操作目标是在稳定供液条件下，把纤维直径控制在目标区间并降低渣球含量。其控制特征是“高转速+高温+高动态响应”，任何机械偏差都会在短时间内放大为产品质量波动。

### 1.3 主要技术参数表

| 参数项 | 标准范围 | 目标意义 |
|---|---|---|
| 主轴转速 | 5500-6500 rpm | 决定纤维细化程度 |
| 设备振动 | ≤3.0 mm/s | 保障机械稳定与安全 |
| 轴承温度 | 40-80℃ | 反映润滑与健康状态 |
| 纤维直径 | 4-7 μm（工艺目标） | 保温性能与强度平衡 |
| 渣球含量 | ≤10%（工艺目标） | 影响成品等级 |
| 供液稳定性 | 连续、无突变 | 保证成纤连续性 |

## 2. 操作规程

### 2.1 开机前检查清单
1. 机械系统：四辊表面、紧固件、联轴器、传动皮带或齿轮箱无异常。
2. 润滑系统：油位、油质、油路压力、油温正常，无渗漏。
3. 冷却系统：冷却水流量与温度达标，回水通畅。
4. 电气与控制：变频器、转速反馈、振动探头、轴温探头在线。
5. 安全装置：防护罩、急停、超振联锁、超温联锁可用。
6. 工艺联锁：确认熔炉出液稳定，供液槽状态满足启动条件。

### 2.2 启动步骤
1. 启动润滑与冷却系统，确认 3-5 分钟稳定运行。
2. 点动检查转向与机械异响，确认后进入低速运行。
3. 按程序升速至工艺初始值（如 5600-5800 rpm），观察振动与轴温。
4. 供液联锁开启，逐步建立稳定甩丝状态。
5. 根据纤维样检结果微调转速与气流参数，进入标准生产窗口。

### 2.3 正常运行监控要点
- 实时监控转速波动，目标保持在设定值±50 rpm 内（内控建议）。
- 振动值超过预警阈值（如 2.5 mm/s）应立刻排查并降负荷观察。
- 轴承温度接近 80℃ 时启动故障处置流程，不得带病高负荷运行。
- 与质检保持联动：纤维直径偏粗或渣球上升时，先看供液与振动，再调转速。
- 班中至少两次检查润滑油状态和回油温度。

### 2.4 停机步骤
1. 按工艺指令逐步降速，避免骤停造成热冲击和机械冲击。
2. 切断供液，保持短时空转清理残留熔体。
3. 转速降至安全值后停主机，继续保持润滑与冷却 10-20 分钟。
4. 停止润滑冷却系统并完成停机点检。
5. 填写停机记录：原因、参数、异常、维修建议。

## 3. 参数标准

### 3.1 参数范围与影响

| 参数 | 标准范围 | 参数偏离影响 |
|---|---|---|
| 转速 | 5500-6500 rpm | 低速易粗纤维与渣球升高；过高速可能断纤、振动上升 |
| 振动 | ≤3.0 mm/s | 超限易造成纤维波动、设备磨损、停机风险 |
| 轴承温度 | 40-80℃ | 温度过高代表润滑或冷却异常，可能引发抱轴 |
| 供液波动 | 尽量平稳 | 波动大会导致直径分布变宽和飞棉增加 |

### 3.2 转速调节与纤维质量关系
- 转速提高通常有利于细化纤维，但必须匹配熔体黏度和供液稳定性。
- 当熔体偏黏时，单纯提速可能导致渣球不降反升，应先处理熔融质量。
- 对同一配方建议建立“转速-纤维直径”经验曲线，按班组固化操作窗口。

### 3.3 振动监测与报警阈值建议
- 正常运行：≤2.0 mm/s；
- 预警区：2.0-2.5 mm/s，需巡检并记录趋势；
- 干预区：2.5-3.0 mm/s，需降负荷并检查动平衡与轴承；
- 报警停机：>3.0 mm/s，按联锁执行保护停机。

### 3.4 参数调整步骤
1. 明确质量问题（粗纤/断纤/渣球/飞棉）并确认监测数据；
2. 优先排除机械异常（振动、轴温、润滑）；
3. 每次调速步长建议 50-100 rpm，小步快审；
4. 调整后观察 5-10 分钟并做纤维抽检；
5. 记录“调整前后数据+质量变化”，形成标准作业经验库。

## 4. 常见故障与处理

1. **故障现象→** 振动值快速升高并伴随异响  
   **可能原因→** 动平衡破坏、轴承损伤、紧固件松动  
   **处理方法→** 立即降速，停机检查并做动平衡校正，更换损坏轴承  
   **预防措施→** 班前紧固巡检，定期做振动趋势分析

2. **故障现象→** 轴承温度持续超过 80℃  
   **可能原因→** 润滑不足、油品劣化、冷却不足或轴承磨损  
   **处理方法→** 检查油路与冷却流量，更换油品或轴承  
   **预防措施→** 严格执行润滑换油周期和油品检测

3. **故障现象→** 纤维直径偏粗且分布不均  
   **可能原因→** 转速偏低、供液不稳、熔体黏度偏高  
   **处理方法→** 适当提速，稳定供液，联动熔炉优化熔融状态  
   **预防措施→** 建立转速-黏度匹配窗口并执行首件确认

4. **故障现象→** 渣球含量突然升高  
   **可能原因→** 熔体未均化、辊面污染、振动超限  
   **处理方法→** 检查熔融段，清理辊面，恢复设备平稳运行  
   **预防措施→** 严控上游熔融质量，按计划清理甩丝部件

5. **故障现象→** 转速无法维持设定值  
   **可能原因→** 变频器参数异常、反馈信号失真、驱动过载  
   **处理方法→** 排查驱动参数与编码器信号，恢复控制闭环  
   **预防措施→** 月度电控点检，关键备件冗余管理

6. **故障现象→** 停机后再次启动困难  
   **可能原因→** 残留熔体附着、机械卡阻、联锁未复位  
   **处理方法→** 清理残留，点动排查卡点，复核联锁条件  
   **预防措施→** 规范停机空转清理和复位流程

## 5. 维护保养

### 5.1 日常保养项目清单
- 检查振动、轴温、转速趋势图，识别早期异常。
- 检查润滑油位与油质，确认油路畅通无泄漏。
- 清理设备表面纤维附着物，保持散热与传感器可用。
- 检查防护罩和急停装置，确保安全可达。

### 5.2 周保养
- 检查联轴器、地脚、关键紧固点扭矩。
- 对振动探头和温度探头进行校验比对。
- 清洁电控柜滤网，检查散热风扇状态。

### 5.3 月保养
- 润滑系统全面检查：油样检测、过滤器更换、油路清洗。
- 检查辊体磨损与偏摆，必要时校正或更换。
- 校验变频器参数与保护阈值，验证联锁动作。

### 5.4 年度检修内容
- 主轴与轴承总成解体检查，必要时整套更换。
- 驱动系统大修：电机绝缘、轴对中、联轴器寿命评估。
- 对机座与基础进行复测，修正共振风险。
- 对整机执行试车验证并更新操作窗口。

### 5.5 轴承润滑周期建议
- 高频运行工况：每 7-14 天补脂或巡检强化；
- 常规工况：按月换油或按运行小时换油（以厂家手册为准）；
- 高温季节与粉尘重工况应缩短 20-30% 保养周期。
