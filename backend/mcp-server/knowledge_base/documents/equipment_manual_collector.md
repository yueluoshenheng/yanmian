# TY-JM120 型摆锤集棉机操作手册

## 1. 设备概述

### 1.1 设备信息
- 设备名称：摆锤集棉机
- 设备型号：TY-JM120
- 制造厂家：TY Forming Systems Co., Ltd.
- 产线位置：离心成纤之后、固化定型之前
- 主要作用：通过负压吸附与摆锤布棉，将纤维均匀沉积到输送网带形成棉层，并与喷胶系统协同完成前成型

### 1.2 工艺定位与功能
TY-JM120 负责把“纤维流”转化为“可固化棉毡”，是决定厚度均匀度、密度分布和后段切割稳定性的关键设备。其核心控制逻辑为“负压稳定+摆锤节奏+网速匹配+喷胶协同”。若负压不稳或滤网堵塞，会直接导致棉层条纹、局部稀薄、边部坍塌等质量问题。

### 1.3 主要技术参数表

| 参数项 | 标准范围 | 控制目标 |
|---|---|---|
| 负压 | -800~-1000 Pa | 稳定吸附与沉积效率 |
| 传送速度 | 8-15 m/min | 匹配产量与厚度要求 |
| 棉层厚度 | 60-120 mm | 满足后续固化压缩窗口 |
| 厚度均匀度偏差 | ≤±5% | 保障产品密度一致性 |
| 摆锤频率（内控） | 按工艺卡设定 | 改善横向铺棉均匀性 |
| 喷胶配合 | 与铺棉同步 | 提升成型和固化质量 |

## 2. 操作规程

### 2.1 开机前检查清单
1. 风机与风道：负压风机运转正常，风道无松脱和严重漏风。
2. 滤网系统：滤网清洁、张紧适中、无破损堵塞。
3. 摆锤机构：驱动电机、连杆、轴承、限位开关状态正常。
4. 输送网带：跑偏装置可用，张力与清洁状态良好。
5. 测厚系统：传感器在线，标定值有效。
6. 喷胶系统：胶液浓度、压力、喷嘴状态、流量计正常。
7. 安全项目：防护栏、急停、联锁、照明、检修门状态确认。

### 2.2 启动步骤
1. 启动风机建立负压，稳定在 -850~-900 Pa 试运行区间。
2. 启动输送网带至低速，确认跑偏与张力正常。
3. 启动摆锤机构，观察摆动幅度和周期稳定性。
4. 与上游离心段联动引入纤维流，逐步提升至目标负荷。
5. 启动喷胶系统并与铺棉节奏同步，调整至目标厚度。
6. 进入稳定生产后开启自动厚度闭环控制。

### 2.3 正常运行监控要点
- 每班确认负压曲线稳定，波动超过设定阈值应立刻排查漏风和堵塞。
- 连续监控棉层厚度与横向偏差，及时修正摆锤参数与网速。
- 观察边部成型，防止两侧偏薄或堆积。
- 关注喷胶雾化与喷幅覆盖，避免条带状含胶不均。
- 做好“参数—质量”记录：负压、网速、厚度、含胶率与后段固化结果。

### 2.4 停机步骤
1. 先降低纤维输入负荷，再逐步降低网带速度。
2. 关闭喷胶并冲洗喷嘴与胶路，防止凝胶堵塞。
3. 停摆锤机构，保持短时负压运行清理浮棉。
4. 停止风机，清理滤网和设备表面附棉。
5. 完成停机点检和交接记录。

## 3. 参数标准

### 3.1 参数范围、影响与调整

| 参数 | 标准范围 | 偏离影响 | 调整方法 |
|---|---|---|---|
| 负压 | -800~-1000 Pa | 负压低：飞棉、沉积差；负压高：棉层压实过度、分布异常 | 通过风机频率、风阀开度联动微调 |
| 传送速度 | 8-15 m/min | 速度快：厚度不足；速度慢：堆积过厚 | 调整网带变频并配合喂棉量 |
| 棉层厚度 | 60-120 mm | 超上限影响固化穿透，低于下限影响成品厚度 | 测厚闭环调节网速与摆锤 |
| 均匀度偏差 | ≤±5% | 超差造成密度不均和强度波动 | 调摆锤频率/幅度，修正风量分布 |

### 3.2 负压调节与棉层均匀度关系
- 负压在目标区间内，纤维捕集效率高，棉层边中差可控。
- 负压偏低时，纤维悬浮增加，形成“薄区与飞棉”；
- 负压偏高时，局部吸附过强，出现“暗带”和密度条纹。
- 推荐以 20-30 Pa 为步长调整，并观察 3-5 分钟厚度曲线再继续。

### 3.3 喷胶系统配合要点
1. 喷胶启动应晚于负压稳定、早于满负荷布棉，避免干棉区。
2. 胶液黏度与喷压需匹配，雾化过粗会造成局部硬块。
3. 含胶率异常时先排查喷嘴堵塞和流量计漂移，再改配方。
4. 喷胶参数调整后必须同步追踪固化率与板材强度。

### 3.4 参数调整步骤
1. 判定问题类型（厚度、均匀度、边部、含胶）。
2. 优先恢复负压稳定，再调网速与摆锤参数。
3. 每次仅调整一个主变量，避免相互干扰。
4. 形成“调整动作—结果”记录并沉淀班组标准。

## 4. 常见故障与处理

1. **故障现象→** 棉层横向条纹明显  
   **可能原因→** 摆锤节奏不稳、风量分配不均、滤网局部堵塞  
   **处理方法→** 校正摆锤参数，平衡风阀，清理堵塞滤网  
   **预防措施→** 周期校验摆锤机构，执行滤网分区清洁

2. **故障现象→** 棉层厚度波动大  
   **可能原因→** 网速波动、测厚漂移、上游供纤不稳  
   **处理方法→** 检查变频器与测厚系统，联动上游稳定供纤  
   **预防措施→** 班前做测厚标定，建立联动巡检机制

3. **故障现象→** 飞棉增多，现场粉尘上升  
   **可能原因→** 负压不足、风道漏风、边封损坏  
   **处理方法→** 提升负压、修复漏点、恢复边封  
   **预防措施→** 每周风道密封检查，关键部位加装观察窗

4. **故障现象→** 喷胶后出现硬块或胶团  
   **可能原因→** 胶液黏度偏高、喷嘴雾化差、胶路污染  
   **处理方法→** 调整配胶浓度，清洗喷嘴和管路  
   **预防措施→** 执行停机冲洗SOP，胶液过滤入系统

5. **故障现象→** 滤网压差持续升高  
   **可能原因→** 纤维堵塞、清灰失效、滤网老化  
   **处理方法→** 停机清理或更换滤网，恢复清灰系统  
   **预防措施→** 设压差预警值并按周点检

6. **故障现象→** 网带跑偏导致边部成型差  
   **可能原因→** 张力不均、纠偏装置失灵、托辊磨损  
   **处理方法→** 调整张力，检修纠偏系统，更换托辊  
   **预防措施→** 班次巡检跑偏趋势，月度做几何校正

## 5. 维护保养

### 5.1 日常保养项目清单
- 清理滤网与设备表面附棉，保持通风路径畅通。
- 检查风机、风阀、负压表与测厚系统状态。
- 检查摆锤连杆、轴承润滑点和异常噪音。
- 检查喷胶系统喷嘴、过滤器、胶路密封。

### 5.2 周保养
- 对滤网执行深度清洁并检查破损区域。
- 检查网带张力、跑偏和纠偏执行器灵敏度。
- 校验负压传感器和测厚仪，修正漂移。

### 5.3 月保养
- 检修风机叶轮与轴承，校验振动和电流。
- 对摆锤机构做间隙测量、紧固和润滑更新。
- 喷胶系统做全流程冲洗，检查计量精度。

### 5.4 年度检修内容
- 风道系统整体检漏与结构修复。
- 摆锤总成大修：轴承更换、传动件寿命评估。
- 网带系统几何校正、托辊与驱动部件升级。
- 测控系统年度标定与联锁功能复验。

### 5.5 滤网清理维护要求
- 正常工况建议每班快速清理、每周深度清理。
- 压差达到预警值时立即处理，不得带堵塞生产。
- 滤网破损应当班更换，避免二次污染和质量波动。
